Git Push - Dealing With Multiple Github Accounts
Like me you also have multiple github accounts. It is a hassle to deal with git push. Below are my way to aleviate the issue:
  1. For each github account, generate an ssh key pair: ssh-keygen -t rsa -C "your github id"
  2. Copy the public key to your github account, rename the private key ~/.ssh/id_rsa to ~/.ssh/id_rsa.githubid
  3. cat > ~/.ssh/config # 1st account Host firstacc HostName github.com User git IdentityFile ~/.ssh/id_rsa.firstacc # 2nd account Host secondacc HostName github.com User git IdentityFile ~/.ssh/id_rsa.secondacc
  4. cd "your 1st project" git config user.email "your email" git config user.name "your name" git add . git commit . -m "init" git remote add origin git@firstacc:your1stProjectGithubId/yourfirstproject.git git push -u origin master
