Git 101- Basic Git Command

I used AWS CodeCommit for the tutorial

  1. Upload public SSH key to AWS IAM user's account under GIT
  2. CodeCommit -> Create a new empty repository
    The repository URL will be: ssh://git-codecommit.us-east-2.amazonaws.com/v1/repos/DemoRepo
  3. Push your code to git repo for the very first time
    1. cd "DemoRepo"
    2. git init
    3. Tell git who you are
      git config user.email "your email"
      git config user.name "your name"
    4. Provide git the address of the remote repository server (origin)
      git remote add origin ssh://git-codecommit.us-east-2.amazonaws.com/v1/repos/DemoRepo
    5. git add .
    6. git commit . -m "init"
    7. Now create a master branch at the remote server then push code to it
      git push -u origin master
  4. Done :)
