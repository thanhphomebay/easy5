Deep Copy: A shallow copy is shallow because it only copies the object but not its child objects. Instead, the child objects refer to the original objectâ€™s child objects. 
 
                # To Verify that both objects really refer to the same object
                print(id(ori))
                print(id(cpy))
 
                import copy
                # Lists inside a list
                ori= [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
 
                # Let's take a shallow copy of ori
                cpy = cpy.copy(ori)
                cpy = ori.copy() #same as above
                cpy = list(ori) # same as above
                cpy = ori[:] #list slicing operator-copy from start to end element, as as above, shallow copy
 
                print(id(ori), id(cpy)) # 140315092110600 140315092266184         
                # ==> It seems as if cpy was a completely independent copy of ori. But it is not.
                print(id(ori[0]), id(cpy[0]))# 140315092110664 140315092110664
                
                # Deep copy
                cpy=copy.deepcopy(ori)
